# 动画快速开始

## 🎯 三种方案,任选其一

### 方案 1: Lottie JSON (推荐 ⭐⭐⭐⭐⭐)

**优点**: 文件最小、加载最快、透明背景、完美循环

**步骤**:
1. 上传视频到: https://lottiefiles.com/tools/video-to-lottie
2. 下载 JSON 文件
3. 放到 `public/girl-animation.json`
4. 完成! 🎉

**文件大小**: ~50KB

---

### 方案 2: GIF 动画 (简单 ⭐⭐⭐⭐)

**优点**: 兼容性好、支持透明背景、实现简单

**步骤**:
```bash
cd baby-integrity-app
ffmpeg -i public/你的视频.mp4 -vf "fps=24,scale=400:-1:flags=lanczos" -loop 0 public/girl-animation.gif
```

**文件大小**: ~2MB

---

### 方案 3: 静态图片 (临时 ⭐⭐)

**优点**: 最简单、无需转换

**步骤**:
1. 已有 `public/girl.png`
2. 无需操作,自动使用

**文件大小**: ~100KB (但无动画)

---
