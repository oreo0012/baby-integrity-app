# 更新日志

## [当前版本] - 2026-01-18

### ✨ 新增功能

#### 1. 动态进度条颜色
- 进度条颜色根据信用分段自动变化
  - 优秀（90-100分）：绿色
  - 良好（60-89分）：黄色
  - 待提高（0-59分）：红色
- 主界面和模态框中的进度条同步更新

#### 2. Toast 通知系统
- 加分时显示绿色成功通知
- 扣分时显示红色警告通知
- 自动3秒后消失，可手动关闭
- 显示具体操作项和分数变化

#### 3. 历史记录功能
- 新增历史记录模态框
- 显示所有加分/扣分操作记录
- 包含操作时间、项目名称、分数变化
- 显示操作前后的分数对比
- 按时间倒序排列（最新的在上面）

#### 4. UI/UX 优化
- 主界面右上角添加历史记录按钮
- 优化模态框视觉效果
- 改进按钮交互反馈
- 统一配色方案

### 🔧 技术改进
- 完善 TypeScript 类型定义
- 优化组件结构和代码复用
- 改进数据管理逻辑
- 添加动画效果（Toast 滑入动画）

### 📝 文档更新
- 更新 README.md 添加详细使用说明
- 添加配置文件说明
