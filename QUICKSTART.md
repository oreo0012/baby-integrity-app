# 快速开始 🚀

## 5分钟上手指南

### 第一步：启动应用
```bash
cd baby-integrity-app
npm run dev
```

访问：http://localhost:5173/

### 第二步：试用功能

#### 1️⃣ 加分操作
- 点击右侧粉色"送花"按钮
- 选择"积极完成学习任务"
- 看到绿色通知："+1分" 或 "+2分" 或 "+3分"（随机）
- 进度条上升，分数增加

#### 2️⃣ 扣分操作
- 点击左侧蓝色"扣花"按钮
- 选择"拒绝完成学习任务"
- 看到红色通知："-1分" 或 "-2分" 或 "-3分"（随机）
- 进度条下降，分数减少

#### 3️⃣ 查看权益
- 点击"我的权益"按钮
- 查看当前等级的权益
- 不同分数段权益不同

#### 4️⃣ 查看历史
- 点击右上角时钟图标
- 查看所有操作记录
- 包含时间、原因、分数变化

### 第三步：观察变化

#### 测试分段变化
1. 初始分数 100 分（优秀等级，绿色进度条）
2. 连续扣分到 89 分以下
3. 进度条变黄色（良好等级）
4. 点击"我的权益"查看权益变化
5. 继续扣分到 59 分以下
6. 进度条变红色（待提高等级）
7. 再次查看权益变化

### 第四步：自定义配置

#### 修改加分项
编辑 `src/config/score-items.json`：
```json
{
  "addItems": [
    {
      "id": "add-custom",
      "name": "自己的加分项",
      "minScore": 5,
      "maxScore": 10
    }
  ]
}
```

保存后刷新页面，点击"送花"即可看到新的加分项。

#### 修改权益
编辑 `src/config/score-ranges.json`，调整各等级的权益数值。

## 🎯 核心概念

### 信用分等级
| 等级 | 分数范围 | 进度条颜色 |
|------|----------|------------|
| 优秀 | 90-100 | 🟢 绿色 |
| 良好 | 60-89 | 🟡 黄色 |
| 待提高 | 0-59 | 🔴 红色 |

### 随机分值
- 每个加分/扣分项都有最小值和最大值
- 系统在范围内随机选择一个值
- 例如："积极完成学习任务" 会随机加 1-3 分

### 权益系统
- 权益根据当前分数实时计算
- 分数越高，权益越好
- 跨越分段时权益自动更新

## 💡 使用场景

### 场景一：每日打卡
早上起床后：
1. 打开应用查看当前分数
2. 如果昨天表现好，家长点击"送花"加分
3. 查看今天的权益（自由时间、零花钱等）

### 场景二：即时反馈
孩子完成作业后：
1. 家长立即打开应用
2. 点击"送花" → "积极完成学习任务"
3. 孩子看到分数增加和绿色通知
4. 获得即时正向反馈

### 场景三：行为纠正
孩子违反约定后：
1. 家长打开应用
2. 点击"扣花" → "违约拖延"
3. 孩子看到分数减少和红色通知
4. 理解行为后果

### 场景四：目标激励
周末前：
1. 孩子查看当前分数（例如 85 分）
2. 发现需要 90 分才能"周末游戏"
3. 努力表现，争取加分
4. 达到目标后获得奖励

## 🔧 常用操作

### 重置数据
打开浏览器开发者工具（F12）：
```javascript
localStorage.clear()
location.reload()
```

### 查看当前数据
```javascript
// 查看当前分数
localStorage.getItem('baby-integrity-current-score')

// 查看历史记录
JSON.parse(localStorage.getItem('baby-integrity-score-history'))
```

### 手动设置分数
```javascript
localStorage.setItem('baby-integrity-current-score', '95')
location.reload()
```

## 📱 移动端使用

### 在手机上访问
1. 确保手机和电脑在同一 WiFi
2. 运行 `npm run dev -- --host`
3. 访问显示的 Network 地址（如 http://192.168.1.100:5173）

### 添加到主屏幕
- iOS: Safari → 分享 → 添加到主屏幕
- Android: Chrome → 菜单 → 添加到主屏幕
