## 🎯 设计理念

### Figma 设计特点
1. **紫灰色调**：营造温馨、柔和的氛围
2. **半透明效果**：增加层次感和现代感
3. **大圆角设计**：友好、可爱的视觉风格
4. **统一的视觉语言**：所有模态框使用相同的配色和布局

### 实现效果
- ✅ 视觉统一性：所有模态框风格一致
- ✅ 层次分明：背景、内容、按钮层次清晰
- ✅ 交互友好：大按钮、清晰的视觉反馈
- ✅ 现代美观：半透明、模糊效果提升质感

## 📊 技术细节

### CSS 类名
```css
/* 模态框背景 */
bg-[#8B7E8F]

/* 选项按钮 */
bg-[#E5DFE6] hover:bg-[#D5CFD6]

/* 底部卡片 */
bg-[#6B6570]/80 backdrop-blur-sm

/* 背景图片 */
opacity-30
```

### 布局结构
```
固定定位容器 (fixed inset-0)
├── 半透明遮罩 (bg-black/40 backdrop-blur-sm)
└── 内容容器 (390x844px)
    ├── 背景色 (#8B7E8F)
    ├── 背景图片（半透明）
    ├── 关闭按钮（右上角）
    ├── 标题（顶部居中）
    ├── 选项列表（中间滚动区域）
    └── 底部信息卡片（半透明）
```
