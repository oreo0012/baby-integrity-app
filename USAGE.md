# 使用指南

## 🎮 如何使用

### 启动应用
1. 打开终端，进入项目目录
2. 运行 `npm run dev`
3. 在浏览器中访问 http://localhost:5173/

### 主界面功能

#### 1. 查看当前信用分
- 主界面中央显示小女孩形象
- 底部卡片显示当前信用分数
- 进度条颜色表示当前等级：
  - 🟢 绿色 = 优秀（90-100分）
  - 🟡 黄色 = 良好（60-89分）
  - 🔴 红色 = 待提高（0-59分）

#### 2. 加分操作
1. 点击"送花"按钮
2. 选择加分原因（如"积极完成学习任务"）
3. 系统自动随机分配分数（在配置的范围内）
4. 显示绿色通知提示加分成功
5. 进度条和分数实时更新

#### 3. 扣分操作
1. 点击"扣花"按钮
2. 选择扣分原因（如"拒绝完成学习任务"）
3. 系统自动随机扣除分数（在配置的范围内）
4. 显示红色通知提示扣分
5. 进度条和分数实时更新

#### 4. 查看权益
1. 点击"我的权益"按钮
2. 查看当前信用等级对应的权益
3. 包括：
   - 每日自由支配时间
   - 每日手表消费额度
   - 每周获得零花钱
   - 周末游戏时间（允许/禁止）

#### 5. 查看历史记录
1. 点击右上角的历史记录按钮（时钟图标）
2. 查看所有加分/扣分操作记录
3. 每条记录显示：
   - 操作类型（加分/扣分）
   - 操作原因
   - 分数变化
   - 操作时间
   - 操作前后的分数

## ⚙️ 自定义配置

### 修改加分项和分值
编辑 `src/config/score-items.json` 文件：

```json
{
  "addItems": [
    {
      "id": "add-1",
      "name": "积极完成学习任务",
      "minScore": 1,
      "maxScore": 3
    }
  ]
}
```

- `id`: 唯一标识符
- `name`: 显示的加分项名称
- `minScore`: 最小分值
- `maxScore`: 最大分值

系统会在 minScore 和 maxScore 之间随机选择一个值。

### 修改扣分项和分值
同样在 `src/config/score-items.json` 文件中修改 `subtractItems` 数组。

### 修改权益配置
编辑 `src/config/score-ranges.json` 文件：

```json
{
  "ranges": [
    {
      "name": "优秀",
      "minScore": 90,
      "maxScore": 100,
      "rewards": {
        "freeTime": {
          "name": "每日自由支配时间",
          "value": 20,
          "unit": "分钟"
        },
        "weekendGame": {
          "name": "周末游戏时间",
          "value": true,
          "unit": ""
        }
      }
    }
  ]
}
```

- `name`: 等级名称
- `minScore`: 最低分数
- `maxScore`: 最高分数
- `rewards`: 该等级对应的权益
  - 数字类型权益：设置 `value` 为数字，`unit` 为单位
  - 布尔类型权益：设置 `value` 为 true/false

## 💡 使用技巧

### 1. 分数管理
- 初始分数为 100 分
- 分数范围：0-100 分
- 达到 100 分后无法继续加分
- 降到 0 分后无法继续扣分

### 2. 权益变化
- 权益根据当前分数实时计算
- 跨越分段时权益会自动更新
- 可随时点击"我的权益"查看当前权益

### 3. 历史记录
- 所有操作都会自动记录
- 记录按时间倒序排列
- 可以追溯所有分数变化

### 4. 数据持久化
- 所有数据保存在浏览器 localStorage 中
- 关闭浏览器后数据不会丢失
- 清除浏览器数据会重置应用

## 🔧 故障排除

### 问题：分数没有更新
- 检查浏览器控制台是否有错误
- 尝试刷新页面
- 检查 localStorage 是否被禁用

### 问题：历史记录为空
- 确认已经进行过加分或扣分操作
- 检查浏览器是否支持 localStorage

### 问题：权益显示不正确
- 检查 `score-ranges.json` 配置是否正确
- 确认分数段没有重叠
- 刷新页面重新加载配置

## 📱 最佳实践

### 家长使用建议
1. 每天固定时间与孩子一起查看信用分
2. 及时记录孩子的好行为和需要改进的地方
3. 定期与孩子讨论权益和目标
4. 根据实际情况调整配置文件

### 孩子使用建议
1. 每天查看自己的信用分和权益
2. 了解哪些行为会加分，哪些会扣分
3. 设定目标，努力提高信用分
4. 查看历史记录，反思自己的行为
