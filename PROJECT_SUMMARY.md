## 🔧 技术架构

### 前端技术栈
```
React 19 + TypeScript
├── Vite (构建工具)
├── Tailwind CSS (样式)
├── Lucide React (图标)
└── localStorage (数据存储)
```

### 项目结构
```
baby-integrity-app/
├── src/
│   ├── components/      # UI 组件
│   ├── config/          # 配置文件
│   ├── lib/             # 工具函数
│   └── styles/          # 样式文件
├── public/              # 静态资源
└── docs/                # 文档（README等）
```

### 数据流设计
```
用户操作 → Modal → data-manager → storage → localStorage
                                      ↓
                                   config.ts
                                      ↓
                              JSON 配置文件
```
